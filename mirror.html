Mirror Transformer Manual


What is the set of all SVG images which can be generated by mirror.py


The mirror transformer will mirror the svg image generated by generate_vine alternating over both the x and the y axis.

In other words, mirror will flip the original image 2n times across the y and x axis depending on the value of n.


This transformer takes one integer parameter ‘n’ where the image will be mirror 2n times. On the condition that n >= 0


Each time the image is mirror alone the y-axis the scale will be ajusted by half to keep the newly formed image on the plane.


If n is an even number the last mirror will be along the x-axis, if n is an odd number the last mirror will be along the y-axis


It mirrors the image alternating over both the y and the x-axis. Starting with the y axis

If n is even it will flip vertically, along the y-axis. If n is odd it will flip horrizontally along the x-axis
