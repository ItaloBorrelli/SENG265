Vine Users Manual

What is the set of all SVG images which can be generated by Vine?

The Vine application creates an image of vines that spiral outward from a central point. There may be any number of vines spiraling out of the central point. Each vine may have sub-vines branching off from it, and those too may have their own sub vines; at each level, a random colour is assigned to the vines. Each sub-vine may be scaled up or down in comparison to it’s parent vine. The angle at which each sub-vine branches off of it’s parent may vary. Within an image, each vine is identical, but vines may vary from one image to another.

In addition, the vines in an image can be duplicated, creating multiple of the same vine within a single image. In addition, vines may have their colours swapped for other colours.

For example:

  • Vine 0 has five branches, each with a branching angle of 0.3 radians. Each branch is sized down by a factor of 0.6, and there are 5 levels of branches. The colours of each branch, from the innermost parent, are Deep Sky Blue, Light Steel Blue, Purple, Blue, and Dark Violet; these colours were assigned randomly.

  • Vine 1 has three branches, each with a branching angle of 0.4 radians. Each branch is sized down by a factor of 0.8, and there are 4 levels of braches. The colours of each branch are Light Sky Blue, Silver, Dark Salmon, and Medium Aqua Marine, all chosen at random. This image is also mirrored twice, once along the x-axis and once along the y-axis, so four identical vines appear in the image.

  • Vine 2 has six branches, each with a branching angle of 0.6 radians. Each branch is sized down by a factor of 0.4, and there are 6 levels of braches. The colours of each branch are Wheat, Orchid, Lawn Green, Dark Green, Slate Gray, and Medium Slate Blue, all chosen at random. This image is also mirrored three times, once along the x-axis and once along the y-axis, so eight identical vines appear in the image.

  • Vine 3 has three branches, each with a branching angle of 0.3 radians. Each branch is sized down by a factor of 0.3, and there are 3 levels of braches. The colours of each branch are Red, Cyan, Dark Salmon, all chosen at random. This image has had a colour swapped to red - Originally, the first colour was Aqua, and then it was changed after generation to red.

What application(s) are supported by Vines?

Vines supports one application: App.sh.

To invoke App.sh from the Linux command line: 

	bash App.sh

App.sh will prompt the user to choose the following options:

  1. Generate a new file, or edit an existing file?
    • If generating a new file:
      1. What should the name of the file be?
      2. How many branches should the vine have?
      3. What is the branching angle in radians?
      4. What scale should each branch have to it's parent?
      5. To what depth should sub-branches be generated?
    • If editing an existing file:
      1. What is the name of the file, excluding any extensions?
  2. Should this vine be mirrored?
    • If yes:
      1. Enter the number of times the image should be mirrored (NOTE: must be greater than zero)
  3. Should this vine have any colours changed?
    • If yes:
      1. What is the the name of the first colour that should be changed?
      2. What is the the name of the colour that it should be changed to? (NOTE: both must be valid colours as listed in css_colours.txt or css_colours_modified.txt)
  4. What should the final output file be named, excluding any extensions?
